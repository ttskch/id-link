# ID Link Chrome Extension

右クリックのコンテキストメニューから要素へのフラグメントリンクをコピーするChrome拡張です。

## 機能

- 右クリックメニューに「この要素へのリンクをコピー」を追加
- id属性またはaタグのname属性がある要素でのみメニューが有効
- 現在のページURL + フラグメント（#id または #name）をクリップボードにコピー

## インストール方法

1. Chromeを開き、アドレスバーに `chrome://extensions/` と入力
2. 右上の「デベロッパーモード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダを選択してインストール完了

## 使用方法

1. 任意のWebページで要素を右クリック
2. id属性またはaタグのname属性がある要素の場合、「この要素へのリンクをコピー」メニューが表示
3. メニューをクリックすると、その要素へのフラグメント付きURLがクリップボードにコピーされます

## 動作確認

`test.html` をChromeで開いて機能をテストできます。

## ファイル構成

- `manifest.json` - 拡張機能の設定
- `background.js` - バックグラウンドスクリプト
- `content.js` - コンテンツスクリプト
- `icons/` - 拡張機能のアイコン
- `test.html` - テスト用ページ
